# Репозиторий с разбором реальных инцидентов из практики IT-поддержки (troubleshooting кейсы).

📂 Содержимое
rdp-session-crash.md - разбор проблемы с падением RDP-сессии (mstsc.exe) из-за драйвера принтера.

🧩 О проекте

В данном репозитории собраны реальные кейсы диагностики и устранения неисправностей в корпоративной инфраструктуре:

 -анализ логов Windows (Event Viewer)
- поиск и определение root cause
- работа с RDP, драйверами и системными компонентами
- применение решений и обходных путей (workarounds)

🛠 Используемые инструменты
-Windows Event Viewer
-Remote Desktop (RDP)
- системные журналы и диагностика ошибок
- базовые инструменты Windows (cmd, поиск, редакторы)

🎯 Цель

Формирование практического опыта в диагностике инцидентов и системном troubleshooting.

📌 Примечание

- Все кейсы обезличены и не содержат конфиденциальной информации.
